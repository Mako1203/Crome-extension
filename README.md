Веб-выделитель (Chrome Extension)

Расширение для браузера, которое позволяет выделять текст на любой веб-странице, подсвечивать его и сохранять в локальное хранилище Chrome вместе с источником (URL, заголовок и время). Используется как инструмент для учёбы и работы с информацией.

Как запустить локально
git clone <ссылка на репозиторий>
cd web-highlighter-extension

Далее:

Открой Google Chrome

Перейди в:

chrome://extensions/
Включи Developer mode
Нажми Load unpacked
Выбери папку проекта
Деплой

Не требуется (расширение устанавливается локально через режим разработчика Chrome).

Что работает
Выделение текста на любой веб-странице
Появление кнопки «Сохранить» рядом с выделением
Подсветка выделенного текста жёлтым цветом
Сохранение выделений в chrome.storage.local
Хранение URL, заголовка страницы и времени сохранения
Popup с отображением всех сохранённых выделений
Возможность удаления сохранённых элементов
Данные сохраняются после перезапуска браузера
Что не доделано

Дополнительные функции (например: категории выделений, экспорт в Markdown, поиск по сохранённым заметкам, восстановление подсветки после перезагрузки страницы) не реализованы, но могут быть добавлены как улучшения проекта.

Что я узнал во время работы

Во время разработки я разобрался с архитектурой Chrome Extensions в Manifest V3, включая разделение на content scripts и popup-окно. Я научился работать с chrome.storage.local для хранения данных между сессиями и понял, как передавать данные из контент-скрипта в интерфейс расширения.

Также я понял, как манипулировать DOM для подсветки текста через Range и Selection API, и как обрабатывать пользовательские события (mouseup, click, scroll) для взаимодействия с интерфейсом прямо на веб-страницах.
